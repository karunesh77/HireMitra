# HireMitra Platform Builder Skill

A comprehensive, production-ready skill for building complete job marketplace platforms from scratch.

## What This Skill Does

This skill guides you through all 7 steps of building a full-stack job platform:

1. **Backend Setup** - Node.js + Express + MongoDB
2. **Database Models** - User, Job, Application schemas
3. **API Routes** - 25+ REST endpoints
4. **Frontend Pages** - Next.js + Tailwind CSS
5. **Advanced Features** - Job creation, dashboards, geolocation
6. **UI Polish** - Professional styling and animations
7. **Deployment** - Vercel + Render

## When to Use This Skill

Use this skill whenever you need to:
- Build a job marketplace platform

- **Frontend**: Next.js 14, Tailwind CSS, Axios
- **Backend**: Node.js, Express.js, MongoDB
- **Authentication**: JWT tokens
- **Database**: MongoDB with Mongoose
- **Deployment**: Vercel (frontend), Render (backend)

